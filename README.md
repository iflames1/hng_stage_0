# HNG Stage 0 API

A simple API that returns basic information for HNG internship.

## API Documentation

**Endpoint**: `GET /`

**Response**:

```json
{

  "current_datetime": "2025-01-30T09:30:00Z",
  "github_url": "https://github.com/yourusername/hng-stage0-api"
}
```
